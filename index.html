Supabase Configuration Required

To load the questions and scenario simulator, connect the application with your Supabase project:

  1. Copy config.example.js as config.js in the project root.
  2. Fill in your credentials (SUPABASE_URL and SUPABASE_ANON_KEY) in config.js.
  3. Create the tables in Supabase by running the script in schema.sql from your console's SQL Editor.
  4. Create the .env file from .env.example and run node migrate_to_supabase.js to populate the database.
